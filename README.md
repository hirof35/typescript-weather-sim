# 2D Weather Simulator

TypeScript と HTML5 Canvas で構築した、インタラクティブな2次元気象図シミュレーターです。

## 🌟 主な機能と特徴
- **気圧配置と風のシミュレーション**: 高気圧・低気圧の座標から気圧傾度力を計算し、コリオリの力を考慮した風ベクトルの合成と粒子アニメーションを実装。
- **Marching Squares（マーチングスクエア法）**: 格子データから等圧線（4hPaごと）をピクセル単位で滑らかに自動描画。
- **前線の自動生成**: グリッド温度の移流（アドベクション）を物理計算し、寒気と暖気が衝突する収束帯（温度傾度の高い場所）に寒冷前線・温暖前線を自動プロット。
- **インタラクティブ操作**: マウスクリックで高気圧・低気圧をリアルタイムに配置・ドラッグ移動・削除可能。
